1.➢¿Que es un Swicht?: Es el que se encarga de crear una red conectando diferentes dispositivos entre ellos.

2.➢¿Que es un Routher?: Establece la conexión entre estas redes.

3.➢¿Que significa WWW?: World Wide Web

4.➢¿Que es un motor de busqueda o navegardor web?: Son sistemas informáticos que trabajan recopilando información en Internet con el objetivo principal de mostrar la información previamente solicitada a los usuarios.

5.➢¿Que es un hosting?: Es un alojamiento web es un servicio donde podrás alojar todos los contenidos necesarios (textos, imágenes, bases de datos, emails…) para que tu web y correo sea accesible por cualquier usuario en internet en cualquier momento.

6.➢¿Que es un dominio?: Es un nombre que funciona para recordar una dirección IP.

7.➢Tipos de dominios:
.com
.org
.net
.int
.edu
.gov
.mil

8.➢¿Que es una Red Social?: Es una página web en la que los usuarios pueden interactuar con personas por medio de mensaje so publicaciones

9.➢¿Que es una Red LAN?: Es un grupo de computadoras y dispositivos periféricos que comparten una línea de comunicaciones común o un enlace inalámbrico a un servidor dentro de un área geográfica específica.

10.➢¿Que es una Red WAN?: Es una red que existe en una zona geográfica amplia. Tu módem envía y recibe información de Internet a través del puerto WAN.

11.➢¿Que es un cable UTP?: Es el que nos permite establecer una conexión de red.

12.➢¿Que es un conector RJ45?: Es un conector que se utiliza para conectar a redes dispositivos mediante un cable que puede ser de hasta 8 hilos en su interior, el conector posee estos 8 pines, aunque no siempre se utilizan todos.

13.➢¿Que es una direccion IP?: Es un conjunto de reglas para la comunicación a través de internet.

14.➢¿Que es un DNS?: Corresponde a las siglas en inglés de "Domain Name System", es decir, "Sistema de nombres de dominio".

15.➢¿Que es Firewall perimetral?: Es el que marca un perímetro entre tu red privada y otras redes de internet.

16.➢¿Que es FQDN?: En sus siglas de Fully Qualified Domain Name, es decir, un nombre de dominio completo.

17.➢¿Que es Ftp?: Es el conjunto de reglas que los dispositivos de una red TCP/IP (Internet) utilizan para transferir archivos.

18.➢¿Que es Hostin Web?: Es un servicio de alojamiento para sitios web.

19.➢¿Que es HTTPS?: Es HyperText Transfer Protocol Secure, protocolo seguro de transferencia de hipertexto) es un protocolo de comunicación de Internet que protege la integridad y la confidencialidad de los datos de los usuarios entre sus ordenadores y el sitio web.

20.➢¿Que es VPN?: Significa red privada virtual.

21.➢¿Que es VPS?: Es la silga en inglés para Virtual Private Server.

22.➢¿Que es Proxy?: Es un equipo informático que hace de intermediario entre las conexiones de un cliente y un servidor de destino.

23.➢¿Que son los Puentes?: Es un dispositivo que realiza interconexiones de red.

24.➢¿Que son las Puertas de Acceso?: Es un puerto es una interfaz a través de la cual se pueden enviar y recibir los diferentes tipos de datos.

25.➢¿Que es Red de Area Local?: Es un grupo de computadoras y dispositivos periféricos que comparten una línea de comunicaciones común o un enlace inalámbrico a un servidor dentro de un área geográfica específica.

26.➢¿Que es TPC/IP?: Es un conjunto de reglas estandarizadas que permiten a los equipos comunicarse en una red como Internet.

27.➢¿Que es WAP?: Es un nuevo estándar de comunicaciones sin hilos que permitirá la utilización de dispositivos móviles para acceder a información, servicios y aplicaciones a través de Internet o de intranets corporativas.
