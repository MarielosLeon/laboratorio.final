1.➢¿Que es un Hardware?: Conjunto de elementos físicos que constituyen en una computadora o sistema.

2.➢¿Que es un Software?: Es un conjunto de programas que no podemos tocar solo ver y que permiten realizar a las computadoras ciertas tareas.

3.➢Tipos de Software: Software de aplicación.
Software de programación.
Software de sistema.

4.➢¿Que es un Sistema Operativo?: Es un programa o conjunto de programas de un sistema informático, que administra los recursos físicos (hardware), los protocolos de ejecución del resto del contenido (software), así como la interfaz de usuario.

5.➢¿Que son los procesadores de palabras?: Es un tipo de programa o aplicación informática cuya función principal es la de crear o modificar documentos de texto, escritos en computadora como Microsoft Word y Word Perfect.

6.➢¿Que son las hojas electrónicas?: Son aplicaciones que permiten efectuar cálculos sencillos y complejos con rapidez y precisión. También sirven para automatizar, utilizando una computadora, como por ejemplo Ecxel.

7.➢¿Que son los software utilitarios?: Son los programas diseñados para o por los usuarios para facilitar la realización de tareas específicas en la computadora.

8.➢¿Que son los Anti-virus?: Son softwares creados para detectar y evitar malware.

9.➢¿Que son los Anti-spam?: Es un método que evita que hayan correos basura.
